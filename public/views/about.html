What is Markeasy?

Markeasy is a unified task grading system. It offers a smart mistakes cataloging for the testing teams, with much ease and fairness.
the system will enhance testing correctness and capabilities using :

  • quick normalization of common mistakes.
  • real time mistakes analysis.
  • display statistics and decency suggestions.

all these will be accompanied by a fast, responsive user interface, to allow testing staff to work as quickly and as fair as possible.
